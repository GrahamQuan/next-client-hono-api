# Next.js 15 App

## Tech Stack

- Next.js 15
- React 19
- Next-intl 4 (Internationalization)
- Next-auth 5 (Authentication)
- Zod (Schema validation)
- Tailwind CSS 4
- Shadcn UI (Components)
- Drizzle ORM
- PostgreSQL (Serverless DB)
- Biome (Linting and formatting)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Authentication

- Next-auth
- Resend
- Google Provider
- Email Provider (6 digit passcode)
